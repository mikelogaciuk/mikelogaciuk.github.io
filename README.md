# mikelogaciuk

My personal blog repository built in Jekyll.

Original [theme](https://github.com/sharadcodes/jekyll-theme-serial-programmer) by [sharadcodes](https://github.com/sharadcodes).

## Build

In order to rebuild blog, type:

```shell
bundle exec jekyll build
```

## Development

To start local build, type:

```shell
bundle exec jekyll serve
```
